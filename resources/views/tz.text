Техническое задание для интернет-магазина по продаже строительных материалов, созданного на Laravel

Введение
1.1. Цель данного технического задания - определить требования и ожидания к разрабатываемому интернет-магазину по продаже строительных магазинов на платформе Laravel.

1.2. Разрабатываемый интернет-магазин должен обеспечивать удобство и функциональность как для покупателей, так и для продавцов.

Общие требования
2.1. Интернет-магазин должен быть создан на платформе Laravel с использованием последних версий фреймворка и сопутствующих пакетов.

5.1. Функция сравнения товаров.

6.1. Сортировка товаров по цене, популярности и новизне.

7.1. Карточка товара должна содержать информацию о названии товара, его описание, цену, изображения, отзывы покупателей и кнопки добавления в корзину и покупки.

8.1. Корзина с возможностью просмотра содержимого, удаления и добавления товаров.



3.1 Регистрация и авторизация пользователей:

Регистрация пользователей должна быть простой и быстрой, с использованием электронной почты.
Авторизованные пользователи должны иметь возможность войти в свой аккаунт, используя свой логин и пароль.
Неавторизованные пользователи должны видеть кнопку “Войти” или “Зарегистрироваться”, чтобы они могли создать аккаунт или войти в систему.
3.2 Возможность создания личного кабинета:

Личный кабинет должен содержать информацию о пользователе, такую как имя, адрес, платежные данные и т.д.
В личном кабинете пользователь должен иметь возможность изменять свои настройки, такие как пароль, адрес доставки и т.п.
Просмотр каталога товаров:

Пользователь должен иметь возможность просматривать каталог товаров, разбитых на категории.
Каталог должен быть организован таким образом, чтобы пользователь мог легко найти интересующий его товар.
Функция поиска:

Должна быть реализована функция поиска товаров по их названию, категории или ключевому слову.
Возможность сравнения товаров:

Для удобства выбора товара пользователь должен иметь возможность сравнивать товары между собой.
Сортировка товаров:

Товары должны быть сортированы по цене, популярности и дате добавления.

В карточке товара должна быть представлена следующая информация:
Название товара;
Описание товара;
Цена товара;
Фотографии товара;
Отзывы других покупателей;
Кнопки “Добавить в корзину” и “Купить”.

2.1 Возможность регистрации и авторизации пользователей:
В интернет-магазине должна быть реализована возможность регистрации пользователей через электронную почту или социальные сети (Facebook, Google, Vkontakte). Также должна быть доступна функция авторизации через уже созданный аккаунт.

2.2 Личный кабинет пользователя:
После успешной авторизации пользователь должен получить доступ к своему личному кабинету. В личном кабинете должна быть доступна информация о пользователе (имя, адрес, контактные данные), а также возможность изменять свои данные и настройки (адрес доставки, платежная информация и т. д.).

2.3 Просмотр каталога товаров:
Пользователь должен иметь возможность просмотра каталога товаров, который будет разбит на категории для удобства навигации. Каталог должен быть оптимизирован для легкого поиска интересующих товаров.

2.4 Функция поиска товаров:
На сайте должна быть реализована функция поиска товаров по их наименованию, категории или по ключевым словам. Результаты поиска должны быть отсортированы в порядке релевантности.

2.5 Функция сравнения товаров:
Для удобства выбора товара пользователю должна быть предоставлена возможность сравнения нескольких товаров между собой по различным параметрам (цена, характеристики, отзывы и т. д.).
